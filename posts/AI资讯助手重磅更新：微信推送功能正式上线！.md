## 🚀 AI资讯助手重磅更新：微信推送功能正式上线！

> 从此告别手动查看，AI资讯自动推送到您的微信！

### 📱 功能亮点

#### 🔥 当天资讯全推送

  * **智能筛选** : 自动筛选当天发布的所有AI资讯
  * **无遗漏** : 不再局限于最新3条，当天所有资讯一网打尽
  * **实时更新** : 每天早上8点定时推送，确保信息及时性

#### 💡 智能推送内容

每次推送包含：

  * 📊 **数据统计** : 总资讯量、新增文章数量
  * 🔥 **当天资讯** : 完整的标题、摘要、分类信息
  * 📅 **时间标识** : 清晰的发布日期和更新时间
  * 🔗 **快速访问** : 一键跳转到完整内容

#### ⚡ 一键配置

  * **零门槛** : 只需一个Server酱密钥
  * **一分钟配置** : 简单3步完成全部设置
  * **开箱即用** : 无需复杂的云服务配置

### 🎯 配置流程

#### 1\. 获取Server酱密钥

    
    
    访问 https://sct.ftqq.com/
    微信扫码登录
    复制推送密钥
    

#### 2\. 配置GitHub Secret

    
    
    仓库设置 → Secrets → Actions
    添加 SERVERCHAN_KEY
    粘贴您的密钥
    

#### 3\. 立即生效

    
    
    # 测试推送
    npm run test-wechat-push YOUR_KEY
    
    # 手动触发完整流程
    GitHub Actions → Run workflow
