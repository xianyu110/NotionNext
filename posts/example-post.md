# 欢迎使用本地Markdown

这是一篇**示例文章**，展示如何在NotionNext中直接使用本地Markdown文件。

## 主要特性

- ✅ 无需Notion数据库
- ✅ 直接编辑Markdown文件
- ✅ 支持frontmatter元数据
- ✅ 更快的构建速度

## 代码示例

```javascript
// 这是一个代码块示例
function hello() {
  console.log('Hello from local Markdown!')
}
```

## 使用方法

1. 在`posts`目录下创建`.md`文件
2. 添加frontmatter配置
3. 编写Markdown内容
4. 重新构建项目

现在你可以直接在本地管理你的文章了！
