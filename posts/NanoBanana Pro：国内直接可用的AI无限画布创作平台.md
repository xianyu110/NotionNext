#### 🎯 什么是 NanoBanana Pro？

**NanoBanana Pro** 是一个基于**无限画布（Infinite Canvas）**概念的非线性 AI
图像创作平台。它打破了传统线性生成的局限，让你在无限延展的数字画布上进行分支创作和思维导图式的创意探索。

**核心理念** ：**每一次生成都不是终点，而是新的起点。** 你可以基于任意结果进行修改、分支或重组，构建属于你的创意宇宙。

#### 🚀 三步开启创作之旅

**第一步：配置 API（关键步骤）**

首次使用需配置 API。无需担心网络问题，我们已预置中转服务：

  1. 点击左上角的 **设置 (Settings)** 图标。
  2. Base URL 保持默认：`https://apipro.maynor1024.live/`
  3. 填入你的 **Google Gemini API Key** （[点此获取](https://aistudio.google.com/app/apikey) 或访问 [中转 Token]() https://apipro.maynor1024.live/console/token
  4. 点击保存，配置完成！

**第二步：挥洒创意**

  1. 在底部输入框描述你的画面（例如：“赛博朋克风格的雨夜街道”）。
  2. （可选）开启左侧 **深度思考** ，让 AI 帮你优化提示词。
  3. 点击发送，稍等片刻，画布上就会诞生你的第一个创意节点！

**第三步：进阶探索**

  * **分支创作** ：选中任意节点，输入新想法，即可自动创建分支节点。
  * **垫图修改** ：对结果不满意？点击节点上的 “参考图” 按钮，基于当前图进行微调。
  * **保存成果** ：满意的创作可导出为 JSON 备份，方便下次继续。

#### ⚠️ 常见问题 (FAQ)

问题| 可能原因| 解决方案  
---|---|---  
**“模型拒绝生成”**|  提示词包含敏感内容或过于像指令| 尝试使用更具体的画面描述，如"背景改为雪山，高清写实风格"。  
**“未接收到图片数据”**|  API 配置错误或网络问题| 确保设置中 Base URL 为
`https://apipro.maynor1024.live/`。  
**“安全策略拦截”**|  触发了 AI 安全审查| 修改提示词，使其更加温和、合规。  
**“403 权限不足”**|  API Key 无效或未开通计费| 检查 API Key 状态，或尝试切换到免费的 Flash 模型。
