### 场景 2：代码重构与优化

**输入**：
```python
# 上传一段性能低下的代码
def find_duplicates(arr):
    result = []
    for i in range(len(arr)):
        for j in range(i+1, len(arr)):
            if arr[i] == arr[j] and arr[i] not in result:
                result.append(arr[i])
    return result
```

**GPT-4o 优化**：
```python
def find_duplicates(arr):
    """
    使用哈希表优化查找重复元素
    时间复杂度：O(n²) → O(n)
    空间复杂度：O(n)
    """
    from collections import Counter
    count = Counter(arr)
    return [item for item, freq in count.items() if freq > 1]

# 性能提升：100 倍（当 n=10000 时）
```
