## 🎬 Sora2 邀请码自助领取系统 - 让AI视频创作触手可及

### 前言

OpenAI 的 Sora2 视频生成模型发布后，引起了广泛关注。作为一款能够生成高质量视频的 AI 工具，Sora2
的能力令人惊叹。然而，由于采用邀请制，很多想要体验的用户苦于找不到邀请码。

为了让更多人能够体验到这一前沿技术，我开发了一个 **Sora2 邀请码自助领取系统** ，完全免费，让大家可以方便快捷地获取邀请码。

### 🌟 系统特点

#### 1\. 完全自助，即时获取

  * 无需等待人工审核
  * 登录后即可立即领取
  * 整个流程不超过1分钟

#### 2\. 安全可靠

  * 使用 GitHub OAuth 认证，保障账号安全
  * 每个用户只能领取一个邀请码，确保公平
  * 部署在 Vercel 云平台，稳定可靠

#### 3\. 资源共享

  * 用完的邀请码可以归还给系统
  * 让更多人受益，形成良性循环
  * 实时显示可用邀请码数量

#### 4\. 用户友好

  * 界面简洁美观，操作直观
  * 详细的错误提示和使用说明
  * 支持移动端访问

### 📱 如何使用

#### 领取邀请码（只需3步）

**第一步：访问系统**  
打开浏览器访问系统网址（部署后的 Vercel 地址）

**第二步：GitHub 登录**  
点击"使用 GitHub 登录"按钮，授权登录（如果你还没有 GitHub 账号，注册一个只需1分钟）

**第三步：领取邀请码**  
登录成功后，点击"立即获取邀请码"按钮，系统会自动分配一个可用的邀请码

就这么简单！你会看到一个6位的邀请码（例如：J8JKM7），妥善保管即可。

#### 归还邀请码

如果你已经使用完邀请码，或者决定不再使用 Sora2，可以将邀请码归还给系统：

  1. 在"归还邀请码"区域输入你的邀请码
  2. 点击"归还邀请码"按钮
  3. 系统会将邀请码重新放入池中，供其他用户使用

**注意** ：只能归还自己领取的邀请码，格式必须是6位字母数字组合。

### 🔐 安全机制

#### 一人一码限制

  * 系统会记录每个 GitHub 用户的领取记录
  * 如果你已经领取过邀请码，再次点击时会显示你之前领取的邀请码
  * 防止恶意囤积，确保资源公平分配

#### 归还权限验证

  * 只能归还自己领取的邀请码
  * 如果尝试归还他人的邀请码，系统会提示错误
  * 保护用户的邀请码不被他人恶意归还

#### 格式验证

  * 邀请码必须是6位字母数字组合
  * 输入框会自动转换为大写
  * 前后端双重验证，确保数据准确性

### 💻 技术实现

这个系统使用了现代化的技术栈：

  * **前端** ：原生 HTML/CSS/JavaScript，轻量高效
  * **后端** ：Vercel Serverless Functions，无服务器架构
  * **数据库** ：Vercel KV（Redis），快速响应
  * **认证** ：GitHub OAuth，安全可靠
  * **部署** ：Vercel，全球 CDN 加速

系统代码简洁高效，支持 KV 存储失败时的内存降级，确保在任何情况下都能正常运行。
