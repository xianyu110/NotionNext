大家好，我是 Maynor！

Seedream 4.0 模型在图像生成质量和对中文的理解上都给我留下了深刻的印象。

为了让更多朋友能轻松体验到它的魅力，我决定动手开发一个简单的 Web 应用，并将其**完全开源** ！

现在，你不需要复杂的环境配置，也不需要了解代码，只需打开浏览器，就能立即开始你的创作。

#### 它有什么功能？

这个小工具的界面非常简洁，核心功能一目了然：

  1. **文生图** ：输入你天马行空的想象，无论是“一只穿着宇航服在月球漫步的猫”，还是“赛博朋克风格的未来城市夜景”，它都能为你生成精美的图像。
  2. **图生图 (可选)** ：如果你有一张图片作为灵感来源，可以上传它作为参考，AI 会在它的基础上进行二次创作，生成风格相似或内容相关的全新图像。
  3. **纯前端，零门槛** ：整个网站只用了 HTML, CSS 和 JavaScript，你可以直接在 GitHub Pages 上使用，无需登录，完全免费。

#### ⚠️ 一个重要的安全提醒

为了能让这个项目以最简单的方式（静态网页）运行在 GitHub Pages 上，我做了一个取舍：将 API 密钥直接写在了前端的 JavaScript
代码中。

**这是一个非常不安全的做法！** 这意味着任何人都可以在你的网页代码里找到这个密钥。

因此，这个项目目前**仅适用于学习和演示目的** 。如果你想基于这个项目进行二次开发，请务必搭建一个后端服务来安全地管理你的 API  
密钥，避免不必要的风险。我放在项目里的密钥也是一个随时可能失效的测试密钥。
